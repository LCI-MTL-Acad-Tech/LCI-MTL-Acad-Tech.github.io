Comment ça marche
1
Les étudiant·e·s exportent leurs journaux

Depuis leur journal de stage, chaque étudiant·e exporte un fichier JSON quotidien et le dépose dans le dossier OneDrive partagé du cours. Les fichiers de configuration (horaire, absences planifiées) y sont aussi.

2
Tu télécharges tous les fichiers d'un coup

Ouvre le dossier OneDrive ci-dessous, sélectionne tous les fichiers (Ctrl+A ou ⌘A), télécharge-les. Pas besoin de trier — cette page identifie automatiquement les étudiant·e·s qui t'ont assigné·e comme superviseur·e.

Ouvrir le dossier OneDrive
3
Tu dépose les fichiers ici

Glisse-dépose tous les fichiers téléchargés dans la zone de dépôt qui apparaîtra après avoir entré ton nom. Tu verras uniquement tes étudiant·e·s, avec leur progression, leurs heures et les éventuelles absences.

💡
Correspondance par nom : le tableau compare ton nom avec le champ « Enseignant·e responsable » que chaque étudiant·e a rempli lors de la configuration. Si un·e étudiant·e n'apparaît pas, c'est probablement parce que son nom ne correspond pas exactement — la page t'indiquera lequel corriger.
